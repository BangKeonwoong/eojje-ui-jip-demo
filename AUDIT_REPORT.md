# 웹사이트 종합 점검 및 개선 보고서

점검일: 2026-07-10

## 결론

현재 사이트는 **공개 프로토타입**으로 배포할 수 있는 수준으로 정리했다. 스토어의 이미지 중심 구성과 절제된 색·여백은 유지했고, 접근성·모바일·신뢰 문구·저장 안전성·배포 보안을 보강했다.

실제 상점 운영은 아직 불가하다. 인증·DB·결제·실제 메시지 발송·서버 권한 검사가 없다.

## 완료한 수정

### 코드·기능

- 손상되거나 구버전인 저장 ID를 검증해 장바구니 전체가 멈추는 오류를 해결했다 (`index.html:2702-2732`).
- 주문 완료 후 새 상품을 담아도 이전 완료 화면만 보이던 재주문 차단을 해결했다 (`index.html:2877-2882`).
- 주문자 정보를 주문 레코드와 함께 보존하고, 로그아웃 시 개인 주문 상태를 삭제하도록 수정했다 (`index.html:2851-2855`, `index.html:3033-3036`).
- 주문 직전에 판매 완료·존재하지 않는 상품을 다시 제외한다 (`index.html:2842-2846`).
- 요청 번호를 고정 날짜+작은 난수에서 현재 날짜+암호학적 난수 기반으로 변경했다 (`index.html:2812-2820`).
- URL 해시, 브라우저 뒤로가기, 화면별 제목, 화면 이동 후 H1 포커스를 추가했다 (`index.html:2715-2796`).
- 동적 이미지 속성을 `sc-camel-src`로 바꿔 렌더 전 `{{p.img}}` 404 요청을 제거했다.

### 보안·개인정보

- 이름·연락처·주문·입고 알림을 영구 `localStorage`에서 탭 단위 `sessionStorage`로 이동했다.
- 관리자 시드 전화번호를 `010-0000-0000`으로 통일해 합성 데이터임을 명확히 했다.
- 해시 기반 CSP, `nosniff`, 클릭재킹 차단, Referrer·Permissions Policy, 관리자 `noindex/no-store`를 추가했다.
- 스토어·관리자 상단에 데모 경계를 명시해 실제 주문·발송·권한으로의 오인을 줄였다.

### 사용성·접근성

- `lang`, 페이지 제목, 설명, OG, favicon, `<main>`, 본문 바로가기를 추가했다.
- 기존 클릭 전용 요소 40개에 버튼 역할·Tab 포커스·Enter/Space 조작을 연결했다.
- 찜 버튼에 이름·`aria-pressed`, 하단 메뉴에 `aria-current`, 토스트에 `aria-live`를 적용했다.
- 검색·모바일 메뉴·라벨 미리보기에 대화상자 역할, 초기 포커스, Tab 트랩, Esc 닫기, 배경 `inert`, 닫기 후 포커스 복원을 추가했다.
- 입력 레이블·자동완성·키보드 유형·길이 제한·전화번호 검증을 보강했다.
- WCAG 대비가 부족했던 보조색·골드 텍스트를 진하게 조정하고 `prefers-reduced-motion`을 적용했다.
- 480px 이하에서 히어로 CTA를 세로 배치하고, 상담 플로팅 버튼을 메뉴 안으로 이동해 가림을 제거했다.
- 가상 조회수·찜 수·과장 표현을 제거하고 데모 지표로 교체했다.

## 남은 우선순위

### P0 — 실서비스 전환 전 필수

1. 서버 인증, `HttpOnly` 세션, RBAC, 주문·매입·문의 API, DB, 재고 잠금을 구현한다.
2. 템플릿 런타임을 사전 빌드로 이전해 `new Function`/`unsafe-eval`과 편집기 `postMessage` 브리지를 제거한다.
3. 실제 사업자 정보·주소·전화·이용약관·개인정보 처리방침·배송·교환·환불 정책을 확정한다.
4. 실제 결제·카카오·사진 업로드·지도의 성공·실패·재시도 흐름을 구현한다.

### P1 — 다음 버전

1. 상품·재고를 하나의 공유 데이터 모델로 통합하고 관리자 변경을 스토어에 즉시 반영한다.
2. 상품 상세에 실제 사진 5~8장, 하자 확대, 소재·연식·중량·분해 여부·출입문 조건·배송 예상 범위를 추가한다.
3. 관리자 표를 시맨틱 `<table>`+모바일 작업 행으로 재구성하고, 상태 순환 클릭을 명시적 선택+확인으로 바꾸다.
4. 외부 이미지·폰트의 사용 권리를 확인하고 자체 호스팅·반응형 이미지로 전환한다.
